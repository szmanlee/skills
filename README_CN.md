# Skills 🛠️

一个全面的开源 AI Agent 技能集合，旨在为 AI 助手提供丰富的功能模块和工具。

## 📋 目录结构

### 🤖 AI Agents (`skills/ai-agents`)
智能体相关的技能模块：
- **agent-council**: 多智能体协调和管理系统
- **agent-identity-kit**: 智能体身份配置工具包 
- **coding-agent**: 代码开发专用智能体
- **idea-coach**: 创意和想法指导助手
- **multi-coding-agent**: 多语言代码开发智能体

### ⚙️ Automation (`skills/automation`)
自动化相关技能：
- 各种工作流自动化工具
- 任务调度和执行模块
- 系统集成自动化

### 💻 Development (`skills/development`)
开发工具相关：
- 代码质量检查
- 项目管理工具
- 开发环境配置

### 🔗 Integration (`skills/integration`)
第三方服务集成：
- API 集成模块
- 外部服务连接器
- 数据同步工具

### 📊 Management (`skills/management`)
管理和监控：
- 性能监控工具
- 资源管理系统
- 日志分析模块

### 🛠️ Utilities (`skills/utilities`)
实用工具集合：
- 文件处理工具
- 数据转换工具
- 通用辅助功能

## 🚀 快速开始

### 安装
```bash
git clone https://github.com/szmanlee/skills.git
cd skills
```

### 使用方式
1. 选择需要的技能模块
2. 根据 skill 目录下的说明文档进行配置
3. 集成到你的 AI Agent 系统中

## 📚 技能规范

每个技能目录都应包含：
- `SKILL.md`: 技能说明文档
- `package.json`: 依赖配置（如需要）
- 主要实现文件
- 示例和测试
