# UI/UX Pro Max Skill

## Description
Advanced UI/UX design and development skill for creating beautiful, intuitive user interfaces and excellent user experiences. Covers design principles, prototyping, accessibility, and modern frontend frameworks.

## Features
- Modern UI design patterns and principles
- User experience research and testing
- Accessibility guidelines and WCAG compliance
- Responsive design and mobile-first approach
- Design systems and component libraries
- Prototyping and wireframing
- User journey mapping
- A/B testing and optimization
- Frontend framework integration (React, Vue, Angular, etc.)

## Usage
Use this skill when you need to:
- Design user interfaces for web or mobile applications
- Improve user experience and usability
- Create responsive and accessible designs
- Build design systems and component libraries
- Conduct user research and testing
- Optimize conversion rates and user engagement

## Tools
- Design tools: Figma, Sketch, Adobe XD
- Prototyping: Figma, InVision, Framer
- Accessibility testing: axe, WAVE, Lighthouse
- Analytics: Google Analytics, Hotjar, Mixpanel
- Frontend frameworks: React, Vue, Angular, Svelte

## Commands
Create a new design component:
```
/ui-ux create-component --name Button --type interactive
```

Analyze UI/UX of existing design:
```
/ui-ux analyze --url https://example.com
```

Generate accessibility report:
```
/ui-ux accessibility-audit --url https://example.com
```

## Best Practices
- Follow WCAG 2.1 AA guidelines for accessibility
- Use semantic HTML elements
- Implement proper color contrast ratios
- Design for keyboard navigation
- Test on multiple devices and screen sizes
- Conduct regular user testing sessions
- Follow mobile-first responsive design principles
